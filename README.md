1. MongoDB 설치

npm install mongodb --save

3. MongoDB 서버 설치

sudo apt-get install mongodb-server
 

4. MongoDB 설정 및 시작

sudo service mongodb restart

6. MongoDB 쉘 시작

mongo
 

7. MongoDB 데이터베이스 선택(생성)

use local
 

8. MongoDB 데이터베이스 조회
